# Simplified-google-search-project
Vytvořit HTML stránku s jedním inputem, kde zadám klíčové slovní spojení. Dostanu vytažené výsledky z google 1. strany (pouze přirozené vyhledávání), které si mohu uložit na PC v jakémkoliv strojově čitelném - strukturovaném formátu (ne HTML). Vytvořený kód pokryt unit testem/y (který minimálně otestuje správnost výstupu).
